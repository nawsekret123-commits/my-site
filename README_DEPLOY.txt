Что загрузить в GitHub:
- app.py
- requirements.txt
- render.yaml
- .gitignore

Что указать в Render -> Environment:
- TELEGRAM_TOKEN = токен бота
- TELEGRAM_CHAT_ID = ваш chat id
- WHATSAPP_NUMBER = 87078340913
- TELEGRAM_USERNAME = ваш_username_без_собаки

Если загружаете через Render по GitHub, render.yaml сам подставит настройки сборки и запуска.
